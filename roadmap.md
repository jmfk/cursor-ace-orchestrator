# ACE Future Roadmap (Post-1.0)

## Phase 10: Scalability & Enterprise Features (Completed)
- [x] **10.1 Distributed Memory**: Implement a distributed vector store for cross-team learning.
- [x] **10.2 RBAC for Agents**: Fine-grained Role-Based Access Control for agent operations.
- [x] **10.3 Enterprise SSO Integration**: Connect ACE to enterprise identity providers.

## Phase 11: Deep LLM Integration (Completed)
- [x] **11.1 Model-Agnostic Orchestration**: Enhanced support for switching between O1, Claude 3.5, and local LLMs.
- [x] **11.2 Fine-tuned Reflection Models**: Train/Fine-tune small models specifically for ACE reflection tasks.
- [x] **11.3 Real-time Context Injection**: Stream context updates to agents during multi-turn sessions.

## Phase 12: Ecosystem & Community (In Progress)
- [ ] **12.1 ACE Plugin Marketplace**: A hub for sharing custom SOPs, Agents, and MDC templates.
- [ ] **12.2 Public API & Webhooks**: Allow external tools to trigger ACE loops and subscribe to events.
- [ ] **12.3 Multi-Language Support**: Expand beyond Python for core ACE library implementations (JS/TS, Rust, Go).
